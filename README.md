## 🚀 Sobre o Projeto

Este projeto é uma **Landing Page de alta performance** desenvolvida para uma clínica de Fisioterapia. O objetivo é unir uma estética limpa (clean) com funcionalidades tecnológicas que facilitam a apresentação de tratamentos e a conversão de novos pacientes.

O site foi pensado para transmitir confiança, profissionalismo e modernidade, destacando a carreira e os diferenciais da **Luana**.

## 🛠️ Tecnologias Utilizadas

Para garantir um visual moderno e código limpo, utilizei:

* ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) - Estrutura semântica.
* ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) - Estilização avançada e Flexbox/Grid.
* ![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black) - Interatividade e dinamismo.
* **Google Fonts** - Tipografia moderna.

## ✨ Funcionalidades

- [x] **Design Responsivo:** Adaptado para celulares, tablets e desktops.
- [x] **Seção de Tratamentos:** Exibição clara dos serviços oferecidos.
- [x] **Área Biográfica:** Espaço dedicado à trajetória profissional.
- [x] **Call to Action (CTA):** Botões estratégicos para agendamento.
